In diesem Ordner sind alle richtigen Seiten der App. Aufgelistet sind alle in Router.svelte.
Das "Routing" funktioniert anders als bei normalen Webseiten, diese "Website" oder App ist als eine
"Single Page Application" oder SPA aufgebaut, es gibt in Wirklichkeit nur genau eine richtige HTML Seite und
alle anderen Seiten werden von Javascript (und der Svelte Bibliothek) dort eingesetzt und mit der
History API wird die URL oben auch angepasst.
